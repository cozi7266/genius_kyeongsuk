7팀  DB + BackEnd 관통 프로젝트
===============================

목차
-------------------------------
1. 개요
    
        이전에 진행했던 프론트엔드 프로젝트에 백엔드와 데이터베이스를 통합하여 로그인, 로그아웃, 회원가입, 회원 탈퇴, 리뷰 작성 등 다양한 기능을 구현한 완성된 운동 플랫폼을 구축했습니다. 이를 통해 사용자들이 효율적으로 운동 콘텐츠를 이용하고, 리뷰를 작성하여 서로의 경험을 공유할 수 있는 사용자 친화적인 환경을 조성했습니다.
